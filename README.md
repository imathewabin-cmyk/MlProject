# Trash Detection Demo

A simple demo website for Teachable Machine image-based trash detection.

## Setup

1. This demo is configured to use the public Teachable Machine model at `https://teachablemachine.withgoogle.com/models/FbooIfkIMq/`.
2. Open `index.html` in a browser.

## Usage

- Click `Start Detection`.
- Allow webcam access when prompted.
- The prediction labels will show class names and probabilities.

## Notes

- This demo uses the CDN versions of TensorFlow.js and the Teachable Machine image library.
- For local testing, some browsers require serving the page via a local web server.
